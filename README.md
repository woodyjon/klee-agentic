# Choisir ses technologies d'agents — EBG, 25 juin 2026

Support de la conférence Klee Group : la présentation (`presentation.html`) et l'atelier interactif qui construit l'architecture d'un agent à partir de 5 questions (`agent-archi-tool.html`). Ouvrez les fichiers `.html` directement dans un navigateur.

## Contenu du dépôt

- **`presentation.html`** — la présentation (slides à dérouler au clavier : `←` `→` pour avancer).
- **`agent-archi-tool.html`** — l'outil interactif : 5 questions → l'architecture de l'agent se construit, avec le service cloud correspondant (AWS / Azure / GCP / on-prem).
- **`Choisir ses technologies d'agents — EBG 25 juin 2026.pdf`** — la présentation en version PDF (à lire / imprimer).
- **`presentation.md`** — le contenu des slides en texte (Markdown).
- **`assets/`** — logos et ressources.
